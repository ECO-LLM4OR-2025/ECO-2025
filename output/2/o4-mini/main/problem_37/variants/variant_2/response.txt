+++++
Sets:
- Products P = {thin, stubby}

Variables:
- x_thin ∈ Integer, ≥ 0: number of thin jars produced
- x_stubby ∈ Integer, ≥ 0: number of stubby jars produced

Parameters:
- shaping_time_thin = 50 (minutes per thin jar)
- shaping_time_stubby = 30 (minutes per stubby jar)
- baking_time_thin = 90 (minutes per thin jar)
- baking_time_stubby = 150 (minutes per stubby jar)
- available_shaping_time = 3000 (minutes per week)
- available_baking_time = 4000 (minutes per week)
- profit_thin = 5 ($ per thin jar)
- profit_stubby = 9 ($ per stubby jar)

Constraints:
- Shaping capacity: 50·x_thin + 30·x_stubby ≤ 3000
- Baking capacity: 90·x_thin + 150·x_stubby ≤ 4000
- Nonnegativity and integrality: x_thin, x_stubby ≥ 0 and integer

Objective:
- Maximize total profit = 5·x_thin + 9·x_stubby
+++++

=====
import gurobipy as gp
from gurobipy import GRB

# Create model
m = gp.Model("terracotta_jars")

# Decision variables
x_thin   = m.addVar(vtype=GRB.INTEGER, name="thin",    lb=0)
x_stubby = m.addVar(vtype=GRB.INTEGER, name="stubby",  lb=0)

# Parameters
shaping_time_thin   = 50
shaping_time_stubby = 30
baking_time_thin    = 90
baking_time_stubby  = 150
avail_shaping       = 3000
avail_baking        = 4000
profit_thin         = 5
profit_stubby       = 9

# Constraints
m.addConstr(shaping_time_thin * x_thin 
            + shaping_time_stubby * x_stubby 
            <= avail_shaping, name="shaping_capacity")

m.addConstr(baking_time_thin * x_thin 
            + baking_time_stubby * x_stubby 
            <= avail_baking, name="baking_capacity")

# Objective: maximize profit
m.setObjective(profit_thin * x_thin 
               + profit_stubby * x_stubby, GRB.MAXIMIZE)

# Solve
m.optimize()

# Save optimal value
if m.status == GRB.OPTIMAL:
    with open('ref_optimal_value.txt', 'w') as f:
        f.write(str(m.objVal))
