Sets:
- Products P = {thin, stubby}

Variables:
- x_thin ∈ Integer, ≥ 0: number of thin jars produced
- x_stubby ∈ Integer, ≥ 0: number of stubby jars produced

Parameters:
- shaping_time_thin = 50 (minutes per thin jar)
- shaping_time_stubby = 30 (minutes per stubby jar)
- baking_time_thin = 90 (minutes per thin jar)
- baking_time_stubby = 150 (minutes per stubby jar)
- available_shaping_time = 3000 (minutes per week)
- available_baking_time = 4000 (minutes per week)
- profit_thin = 5 ($ per thin jar)
- profit_stubby = 9 ($ per stubby jar)

Constraints:
- Shaping capacity: 50·x_thin + 30·x_stubby ≤ 3000
- Baking capacity: 90·x_thin + 150·x_stubby ≤ 4000
- Nonnegativity and integrality: x_thin, x_stubby ≥ 0 and integer

Objective:
- Maximize total profit = 5·x_thin + 9·x_stubby
