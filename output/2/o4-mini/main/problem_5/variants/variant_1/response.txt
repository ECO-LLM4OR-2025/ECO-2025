+++++
Sets:
- Products P = {hardwood, vinyl}

Variables:
- x_h: continuous, production quantity of hardwood (square feet)
- x_v: continuous, production quantity of vinyl (square feet)

Parameters:
- profit_h = 2.5  (profit per square foot of hardwood)
- profit_v = 3.0  (profit per square foot of vinyl)
- demand_h_min = 20000   (minimum hardwood demand)
- demand_v_min = 10000   (minimum vinyl demand)
- ship_min = 60000       (minimum total shipping requirement)
- prod_h_max = 50000     (maximum weekly hardwood production)
- prod_v_max = 30000     (maximum weekly vinyl production)

Constraints:
1. Hardwood demand:      x_h ≥ demand_h_min
2. Vinyl demand:         x_v ≥ demand_v_min
3. Total shipped:        x_h + x_v ≥ ship_min
4. Hardwood capacity:    x_h ≤ prod_h_max
5. Vinyl capacity:       x_v ≤ prod_v_max
6. Nonnegativity:        x_h ≥ 0,  x_v ≥ 0

Objective:
- Maximize total profit = profit_h * x_h + profit_v * x_v
+++++
=====
import gurobipy as gp
from gurobipy import GRB

# Create model
model = gp.Model("flooring_production")

# Decision variables
x_h = model.addVar(lb=0, ub=50000, name="hardwood")  # Hardwood sq ft
x_v = model.addVar(lb=0, ub=30000, name="vinyl")     # Vinyl sq ft

# Constraints
model.addConstr(x_h >= 20000, name="min_hardwood")       # Min hardwood demand
model.addConstr(x_v >= 10000, name="min_vinyl")          # Min vinyl demand
model.addConstr(x_h + x_v >= 60000, name="min_total")    # Shipping requirement

# Objective: maximize profit
model.setObjective(2.5 * x_h + 3.0 * x_v, GRB.MAXIMIZE)

# Optimize the model
model.optimize()

# Write optimal value to file
if model.status == GRB.OPTIMAL:
    with open("ref_optimal_value.txt", "w") as f:
        f.write(str(model.objVal))
